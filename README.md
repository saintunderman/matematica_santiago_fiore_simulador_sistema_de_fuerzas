# Simulador de Sistemas de Fuerzas

Este proyecto es una herramienta educativa interactiva diseñada para visualizar **sistemas de fuerzas concurrentes** y analizar el **equilibrio estático** en un plano bidimensional. Desarrollado por **Santiago Fiore**, el simulador permite a estudiantes y docentes experimentar con vectores de forma intuitiva y precisa.

---

## Características Principales

* **Interactividad Total**: Permite activar o desactivar hasta 3 fuerzas independientes.
* **Cálculos Instantáneos**: Calcula automáticamente la magnitud y el ángulo de la **Fuerza Resultante** ($$F_R$$) y la **Fuerza Equilibrante** ($$F_E$$).
* **Visualización de Componentes**: Opción para descomponer cada vector en sus ejes rectangulares ($F_x$ y $F_y$).
* **Indicador de Equilibrio**: Notificación visual dinámica que indica si el sistema se encuentra en equilibrio estático ($$\sum F = 0$$).
* **Gráfico Ampliado**: Función de zoom para visualizar el diagrama de cuerpo libre en pantalla completa.
* **Alto Rendimiento**: Renderizado optimizado a 60 FPS mediante `requestAnimationFrame` y soporte para pantallas HiDPI/Retina.

---

## Tecnologías Utilizadas

* **HTML5 & CSS3**: Diseño responsivo y moderno con aceleración por hardware (`will-change`, `transform`).
* **Vanilla JavaScript (ES6+)**: Lógica del simulador encapsulada en clases para mayor limpieza y eficiencia.
* **Canvas API**: Renderizado gráfico de vectores y cuadrículas dinámicas.
* **MathJax 3**: Renderizado de fórmulas matemáticas.

---
