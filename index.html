📋 Instrucciones de Uso

Este simulador permite visualizar sistemas de fuerzas concurrentes y calcular la fuerza resultante y equilibrante.

  • Activar/desactivar fuerzas: Usa el interruptor junto a cada fuerza para activarla o desactivarla.
  • Ajustar parámetros: Modifica la magnitud (en Newtons) y el ángulo (en grados) de cada fuerza activa.
  • Visualización: Observa en tiempo real cómo cambian los vectores de fuerza, la resultante y la equilibrante.
  • Análisis de equilibrio: El indicador te mostrará si el sistema está en equilibrio estático.
  • Componentes vectoriales: Activa "Mostrar componentes" para visualizar las componentes X e Y de cada fuerza.
  • Limpiar: El botón "Limpiar Todo" restablece los valores a cero.
  • Ampliar gráfico: Usa el botón "🔍 Ampliar Gráfico" para ver el diagrama en pantalla completa.

📚 Fundamentos Teóricos

Un sistema de fuerzas concurrentes es un conjunto de fuerzas cuyas líneas de acción se cruzan en un único punto. Para analizar el efecto combinado de estas fuerzas, utilizamos el Principio de Superposición.

\[ \vec{F}_R = \sum_{i=1}^{n} \vec{F}_i = \vec{F}_1 + \vec{F}_2 + \vec{F}_3 + \dots \]

Descomposición en Componentes

\[ F_x = |\vec{F}| \cos(\theta) \quad \text{y} \quad F_y = |\vec{F}| \sin(\theta) \]

Cálculo de la Fuerza Resultante

\[ F_{Rx} = \sum F_{ix}, \quad F_{Ry} = \sum F_{iy} \]
\[ |\vec{F}_R| = \sqrt{F_{Rx}^2 + F_{Ry}^2}, \quad \theta_R = \operatorname{atan2}(F_{Ry},F_{Rx}) \]

Condición de Equilibrio

\[ \vec{F}_R = 0 \iff \sum F_x = 0 \ \text{y}\ \sum F_y = 0 \]

Fuerza Equilibrante

La fuerza equilibrante es una fuerza que, aplicada al mismo punto que las demás, produce el equilibrio del sistema. Tiene la misma magnitud que la resultante pero dirección opuesta:

\[ \vec{F}_E = -\vec{F}_R \]
